# FishOS

A fish-themed webOS built for the WebOS mission on Stardance. Productivity? What's that? This in-browser operating system is designed to be as distracting as possible. There is always something in the way. You will never finish that essay. There are so many goldfish.

Most of the images will do something if clicked, so keep that in mind as you go through the site. At the moment, there is one big, very annoying bug that keeps all windows except the welcome window from opening and closing. I'm working on it, but I haven't managed to track down the problem yet. If your browser does not support google fonts, the site will use system fonts instead.
